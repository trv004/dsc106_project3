How Does My Health Compare to Patients Like Me?

An Interactive Exploration of Lab Results by Age, Gender, Height, and Weight Among Surgical Patients.

We created an interactive radar chart that helps us compare and understand how our blood work stacks up against the average lab values, according to our age, height, weight, and gender. We wanted the visualization interface to feel personal and have some exploratory elements so that you can see the typical blood results for someone with similar demographics as you, or just play with the sliders and dropdown and see some patterns within the blood results. When filtering using the sliders and dropdown, the radar chart recomputes the average results for the selected demographic, so you can see the snapshot for that specific demographic and compare it to others or yourself.

In designing this radar chart, we grouped all individual lab tests into five categories (Blood Cells & Inflammation, Electrolytes & Metabolic Panel, Liver & Protein Function, Coagulation & Blood Gases, and Kidney & Waste Clearance). Each test is then scaled between 0 and 1 based on its published reference range, allowing us to normalize and directly compare scores across tests.

=======

Note: Lab test results were normalized using reference ranges specific to each test. For each result, the normalized value was calculated as:


  Normalized Value = (Result - Reference Min) / (Reference Max - Reference Min)

Values are scaled between 0 and 1, where:

  • 0 represents the lower bound of the normal reference range,
  • 1 represents the upper bound,
  • Values outside the reference range are capped at 0 or 1 and do not indicate how far out of range the original value was.

This normalization allows for comparability across different lab tests. For each user-defined group (based on age, gender, height, and weight), normalized values were averaged to produce a composite score for each test.
